## 2.2 CPU的那些事儿

### 2.2.1 从CPU联系到 Java

笔者个人记录：

按照书本知识发现javap提示文件不存在，后来发现是要先使用 javac 编译代码，在编译之后才可以使用 javap 来查看代码。自己落了一步而已。

操作步骤应该为:

`javac testClassSum.java   `

`javap -verbose testClassSum`

注意：此处如果文件编码是 GB2312，那么会报错其编码不是 UTF-8，导致文件无法解析（在 javac 就报错）
