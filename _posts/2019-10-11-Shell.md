# 1. Shell 简介和 Shell 环境

Shell 既是一种命令语言，又是一种程序设计语言。

Shell 提供了一个应用程序，其提供了一个界面，用户通过这个界面进行**访问操作系统内核** 的操作。

## 1.1 Shell 环境

本教程之中所说的 Shell 指的是 Bash， Bash 是大多数系统的默认 Shell， 也是大部分 Linux 的默认shell。

```bash
#！
```

就是告诉系统，其之后的路径所指定的程序就是解释这个脚本文件的 Shell 程序。
